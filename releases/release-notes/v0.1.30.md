# PiPLC v0.1.30 Release Notes

Release: v0.1.30
Date: 2026-02-14

## Summary

- Added an HMI resolution outline to visualize the target display area on the canvas.
- Added a Canvas Size dialog with Raspberry Pi and standard display presets, plus custom sizing.
- Runtime mode now auto-fits the view to the target resolution, while editor mode keeps expanded design margins.
