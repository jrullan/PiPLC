# PiPLC v0.1.28 Release Notes

Release: v0.1.28
Date: 2026-02-14

## Summary

- Added a branded startup splash screen that displays app name, version, and author.
- Fixed a shutdown deadlock where the process could remain running after window close.
- This fix prevents `PiPLC.exe` from staying locked and blocking installer upgrades.
