# PiPLC v0.1.29 Release Notes

Release: v0.1.29
Date: 2026-02-14

## Summary

- Added HMI fullscreen kiosk mode (`F11`) to hide UI chrome during operation.
- Fullscreen behavior is safer: `F11` or `Escape` exits fullscreen, and `Alt+F4` exits fullscreen first.
- Splash screen and About dialog now show consistent branding and author information.
