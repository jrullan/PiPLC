# PiPLC v0.1.25 Release Notes

Release: v0.1.25
Date: 2026-02-10

- Remote contexts can disconnect without stopping the engine, with a warning if routes still reference the context.
- Raspberry Pi/Linux packaging now includes I/O provider plugins reliably (Explorer HAT Pro fix included).
- HMI properties UI no longer shows the internal ID field.
