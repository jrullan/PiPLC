# PiPLC v0.1.25 Release Notes

Release: v0.1.25
Date: 2026-02-10

## Highlights

- Remote context tabs now support disconnecting without stopping the remote engine.
- Linux/RPi packaging now reliably ships the I/O provider plugins, including Explorer HAT Pro.

## New Features

- Remote context tab right-click menu includes "Disconnect" to detach from a remote engine without stopping execution.
- Warns if signal routes reference the context being disconnected.

## Improvements

- Removed the ID field from the HMI Properties panel (internal-only field).

## Fixes

- Explorer HAT Pro plugin RTTI symbol error fixed by linking against `piplc-core`.
- Docker ARM64 build now deploys I/O provider plugins into the .deb package.
