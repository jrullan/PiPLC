# PiPLC v0.1.21 Release Notes

Release: v0.1.21
Date: 2026-02-08

## Highlights

- Modbus TCP Server support in the engine for external client access.

## New Features

- Modbus TCP Server in `piplc-engine` to expose PLC memory to external clients.
- Command-line options:
  - `--modbus-server` to enable the server
  - `--modbus-port` (default 502)
  - `--modbus-unit-id` (default 1)
- Default mapping:
  - I:0 -> discrete inputs
  - O:0 -> coils
  - N:0-7 -> input registers
  - N:10-17 -> holding registers
