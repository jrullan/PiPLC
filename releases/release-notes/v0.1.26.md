# PiPLC v0.1.26 Release Notes

Release: v0.1.26
Date: 2026-02-11

## Summary

- HMI can now follow the IDE context or stay pinned to a selected context for independent monitoring.
- Runtime diagnostics now include scan count and average scan cycle time in the HMI status bar.
- HMI runtime safety improved by auto-returning to editor mode when program downloads replace the monitored context.
- Core app lifecycle was refactored for cleaner startup and independent IDE/HMI live monitoring.
