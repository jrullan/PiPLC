# PiPLC v0.1.13 Release Notes

Release: v0.1.13
Date: 2026-02-06

## Highlights

- HMI usability polish and icon updates.

## New Features

- SVG icons for all 13 HMI element types in the element palette.
- SVG icons for HMI toolbar actions (Editor Mode, Runtime Mode, Show Grid, Snap to Grid).
- New, Open, and Save toolbar buttons in the HMI window for quick file operations.

## Improvements

- Properties panel now hides controls when no element is selected.
- Grid visibility now respects saved project settings when opening files.

## Fixes

- HMI standalone app now loads icons correctly.
- New HMI projects no longer incorrectly prompt to save on open/exit without changes.
- Grid toggle in View menu now correctly shows/hides the grid.
