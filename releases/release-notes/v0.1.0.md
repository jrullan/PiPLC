# PiPLC v0.1.0 Release Notes

Release: v0.1.0
Date: 2026-02-04

## Highlights

- First public release of PiPLC for end users.

## New Features

- Visual ladder logic editor with drag-and-drop rungs and branches.
- Contact decorators for timers, counters, one-shots, and debounce filters.
- Virtual I/O simulation for testing without hardware.
- Remote mode support for connecting the editor to a Raspberry Pi engine.
- GPIO mapping support for real hardware I/O.

## Improvements

- Initial packaging for Windows and Raspberry Pi (ARM64).
