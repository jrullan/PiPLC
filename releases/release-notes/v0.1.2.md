# PiPLC v0.1.2 Release Notes

Release: v0.1.2
Date: 2026-02-04

## Highlights

- Added Explorer HAT Pro I/O provider support.

## New Features

- New instructions: BSL (Bit Shift Left) and BSR (Bit Shift Right).
- New instruction: N2B and expanded word-address support across instructions.
- Explorer HAT Pro I/O provider for Raspberry Pi projects.

## Improvements

- Word addressing support extended across instruction set.

## Fixes

- Force simulation behavior corrected.
- Branch OTE execution fixed.
- Cursor positioning improvements in the editor.
